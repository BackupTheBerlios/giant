\section {Compiler und Bibliotheken}
Das System soll in der Sprache Ada95 mit GNAT 3.14 entwickelt werden, wobei GTK/ADA 1.2.12 als grafische
GUI-Bibliothek eingesetzte werden soll. Das System baut auf der vom Kunden bereit gestellten IML-Graph-Bibliothek auf 
und soll des weiteren zur Unterstützung der Wartbarkeit möglichst auch die vom Kunden zur Verfügung gestellten 
Datenstrukturen (wie z.B. Hashtables aus Bauhaus/reuse/src) nutzen.
