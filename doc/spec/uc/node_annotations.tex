\begin{uc}[Fenster-Knoten annotieren]{UC: Fenster-Knoten annotieren}
  Dieser UseCase dient zum Erzeugen, Ändern und Anschauen 
  von Knoten-Annotationen.
  Er erstellt eine neue Annotation für einen Fenster-Knoten, falls dieser
  noch nicht annotiert ist.\\
  Ist der Fenster-Knoten schon annotiert, wird diese Annotation angezeigt 
  und kann vom Benutzer geändert werden.
  
  
  \begin{precond}
    \cond Das Programm ist gestartet.
    \cond Ein Projekt ist geladen.
    \cond Es gibt mindestens ein Anzeigefenster mit Fenster-Knoten.
  \end{precond}

  \begin{postsuccess}
    \cond Die neue Annotation oder die Änderungen sind noch nicht
          in der Verwaltungsdatei für Knoten-Annotationen eingetragen.
    \cond Der annotierte Knoten wird in allen Anzeigefenstern als annotiert
          hervorgehoben (hat nun das entsprechende Icon).
	  
  \end{postsuccess}

  \begin{postfail}
    \cond Das System und die Verwaltungsdatei für Knoten-Annotationen
          bleiben im bisherigen Zustand.
  \end{postfail}
  
  \begin{proc}
    \step[1] 
    Der Benutzer wählt den Fenster-Knoten aus, der annotiert werden soll und
    öffnet das entsprechende POPUPMENÜ -> VERWEIS ZU GUI.
    \step[2]
    GIANT zeigt nun den EINGABEDIALOG FÜR ANNOTATIONEN. Existiert bereits eine
    Annotation so wird der entsprechende Text angezeigt, anderenfalls wird
    nichts angezeigt.
    
    
    \step[3a]
    Falls ein Knoten neu annotiert werden soll oder falls die Annotation zu
    einem bestehenden Knoten geändert werden soll, gibt der Benutzer
    Im EINGABEDIALOG FÜR ANNOTATIONEN GIBT den entsprechenden Text ein.
    \step[3b]
    Will sich der Benutzer eine bestehende Knoten-Annotation nur auschauen,
    so tut er dies jetzt, ändert aber den Text nicht. 

    \step[4]
    Nach Abschluss der Aufgabe betätigt der Benutzer den \gq{OK Button}
            
    \step[5]
    GIANT übernimmt die vorgenommenen Änderungen. Hat der Benutzer allen Text
    zu einer bestehenden Annotation entfernt, so wird die Knoten-Annotation
    gelöscht. 
    


  \end{proc}

  \begin{aproc}
    \astep{2} Der Benutzer bricht die Eingabe der Annotation mit Cancel ab.

  \end{aproc}
\end{uc}

% ==============================================================================


% ==============================================================================

\begin{uc}[Label]{UC: Knoten-Annotation löschen}
Löscht eine bestehende Knotenannotation.
\end{uc}

% ==============================================================================

\begin{uc}[Label]{UC: Knoten-Annotationen filtern}
>>>> EVENTUELL
Entfernt alle Knoten-Annotationen, deren zugehlörige Knoten nicht mindestens
in einem Anzeigefenster visualisiert sind.
\end{uc}
