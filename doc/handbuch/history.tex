\section*{Versionsgeschichte}

\item Version 1.0

  Diese Version wurde mit dem Kunden durchgesprochen,
  nötige Änderungen wurden festgehalten.

  Änderungen laut Kundenwunsch und Anpassungen an
  geänderte/wegefallene Funktionalitäten durchgeführt.
  Änderungen und Korrekturen laut internem Review durchgeführt.

  Änderungen an hinzugekommener Funktionalität durchgeführt.

\item Version 1.3 (23.09.03)

Version vor der endgültigen Version des Handbuches, kleinere Korrekturen
durchgeführt, Index aktualisiert.\\

durchgeführt werden.
